## III. The Frontier

### What We Don't Know Yet

**1. The Derivation Question**

Can we actually derive the specific laws of physics (Standard Model, General Relativity) purely from constraint-preservation requirements?

Not: "Given these fields and these symmetries, what physics emerges?"
But: "Given only 'constraint structures must be self-consistent,' what physics MUST emerge?"

This is the hard problem. NMSI takes a step toward it. We've taken further steps. But nobody has completed it.

**2. The Consciousness Formalization**

We said: Consciousness is what constraint-preservation feels like from the inside.

But: Can we formalize this? Can we say precisely which constraint structures have "insides"? Is there a threshold? A spectrum?

This connects to IIT (Integrated Information Theory) but goes beyond it. IIT measures phi (integration). We're asking about the intrinsic character of constraint-preservation.

**3. The Uniqueness Question**

Is our universe the ONLY self-consistent constraint structure? Or one of many?

If many: Why this one? (Anthropic? Random? Necessary?)
If only one: What makes alternatives inconsistent?

This connects to Tegmark's Mathematical Universe, but we're asking different questions.

**4. The Empirical Tests**

What would CONFIRM this framework? What would FALSIFY it?

If everything is constraint structure, how would we ever see otherwise? Is this unfalsifiable?

(Note: Unfalsifiable isn't automatically bad. Math is unfalsifiable. Logic is unfalsifiable. If this framework is at the same level as math/logic, unfalsifiability is appropriate.)

### The Most Important Question

"What is X made of?" — Wrong question

"What is the substrate?" — Wrong question
