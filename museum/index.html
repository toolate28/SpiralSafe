Gallery 1: Basic Logic Gates

Dimensions: 50×6×15 blocks | Complexity: Beginner | Interactive: Yes

Learn how computers think! This exhibit demonstrates AND, OR, NOT, and XOR gates using redstone circuits. Each gate includes:

  • Interactive levers (inputs)
  • Redstone lamp (output)
  • Educational signage
  • NPC guide (Professor Redstone)
Input A + Input B = Output
AND:  Both ON  → ON
OR:   Either ON → ON
NOT:  Inverts signal
XOR:  Different → ON

Gallery 2: Binary Counter

Dimensions: 12×5×10 blocks | Complexity: Intermediate | Interactive: Yes

Watch a 4-bit binary counter count from 0 to 15! Features:

  • 4 redstone lamps (bits)
  • Adjustable clock speed
  • Binary/Decimal comparison chart
  • NPC guide (Ada Lovelace)
0000 = 0  |  0100 = 4  |  1000 = 8  |  1100 = 12
0001 = 1  |  0101 = 5  |  1001 = 9  |  1101 = 13
0010 = 2  |  0110 = 6  |  1010 = 10 |  1110 = 14
0011 = 3  |  0111 = 7  |  1011 = 11 |  1111 = 15
